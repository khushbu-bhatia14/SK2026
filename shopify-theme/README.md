# SadaKappda Shopify Theme - Setup Guide

## Theme Overview
A premium, fully dynamic Shopify OS 2.0 theme for the **SadaKappda** Indian D2C apparel brand. Features:
- **"Quietly Premium"** design philosophy
- Switchable color modes (Light/Dark/Mixed) from Theme Editor
- Fully customizable via Shopify Theme Editor
- Mobile-first responsive design
- Indian D2C trust signals (COD, Easy Returns, Secure Payments)

## Quick Installation

### 1. Upload Theme to Shopify
1. Log in to your Shopify Admin
2. Go to **Online Store → Themes**
3. Click **"Add theme" → "Upload zip file"**
4. Select `sadakappda-theme.zip`
5. Wait for upload to complete

### 2. Publish Theme
1. Once uploaded, click **"Publish"** on the theme card
2. Confirm to make it your live theme

### 3. Configure Theme Settings
1. Click **"Customize"** on your theme
2. Go to **Theme Settings** (gear icon in left sidebar)

#### Essential Settings to Configure:
- **Logo & Favicon**: Upload your SadaKappda logo (included in brand-assets folder)
- **Color Mode**: Choose Light/Dark/Mixed
- **Primary Color**: Default is orange (#FF6B35)
- **Secondary Color**: Default is magenta (#E91E8C)
- **Contact Email**: reachus@SadaKappda.com
- **WhatsApp Number**: Add when ready

### 4. Set Up Navigation
1. Go to **Shopify Admin → Online Store → Navigation**
2. Edit "Main menu" and add:
   - Men
   - Women
   - Best Sellers
   - New Arrivals
   - Sale / Offers
   - Size Guide
   - Track Order
   - Contact

### 5. Create Required Collections
Create these collections in Shopify:
- Hoodies
- Sweatshirts
- Men
- Women
- Best Sellers
- New Arrivals
- Sale

### 6. Create Required Pages
Go to **Online Store → Pages** and create:
- **About Us** (select "page.about-us" template)
- **Contact** (select "page.contact" template)
- **Size Guide** (select "page.size-guide" template)
- **FAQ** (select "page.faq" template)
- Returns & Refund Policy
- Shipping & Delivery Policy
- Privacy Policy
- Terms & Conditions

### 7. Upload Brand Assets
The `brand-assets` folder contains:
- `SadaKappda-Logo.png` - Your brand logo
- `CozyBear-Hoodie.jpg` - Product mockup
- `SugarStrength-Hoodie.jpg` - Product mockup
- `Model-Shot-1.jpg` - Model wearing product

Upload these via:
1. **Logo**: Theme Settings → Logo & Favicon
2. **Hero Images**: Home Page → Hero Slider → Edit slide → Background image
3. **Products**: Add to product listings

## Theme Sections (Homepage)

The homepage includes these sections (all editable):
1. **Hero Slider** - Dynamic sliding blocks with image/video backgrounds
2. **Trust Badges** - COD, Easy Returns, Fast Shipping, Secure Payments
3. **Category Blocks** - Shop Men/Women cards
4. **Best Sellers** - Featured collection
5. **Brand Story** - Why SadaKappda section
6. **New Arrivals** - Featured collection
7. **Testimonials** - Customer reviews
8. **Newsletter** - Email subscription

All sections can be:
- Reordered
- Edited
- Hidden/shown
- Customized via Theme Editor

## Color Mode Options
In Theme Settings → Color Mode:
- **Light**: White background, dark text throughout
- **Dark**: Black background, white text throughout
- **Mixed** (Default): Dark header/footer, light body

## Trust Badges
Enabled by default in Theme Settings → Trust Badges:
- COD Available
- Easy Returns
- Fast Shipping
- Secure Payments

## Product Cards
Settings in Theme Settings → Product Cards:
- Color swatches
- Hover second image
- Quick add button
- Sale/New badges

## Technical Details
- **Framework**: Shopify OS 2.0 (Liquid + JSON templates)
- **CSS**: Custom CSS with CSS variables
- **JavaScript**: Vanilla JS (no framework dependencies)
- **Fonts**: Josefin Sans (headings), Assistant (body) - loaded via Shopify fonts
